# 导航

- 本部分, 只包含 Rust 最基础的语法点.
- 涉及到高级语法点(宏, trait), 也只会简单介绍, 不会展开.
- 进阶语法, 会在 `Rust 进阶语法部分` 详细介绍. 不用担心.


## 目标

- 掌握此部分, 可以读懂 rust 开源项目代码 80% 代码.
- 以此, 可以参与改改 社区代码, 而不具备独立写新项目能力.
  - 这个小目标, 是有意义的.
  - 很多非 rust 开发者, 偶尔也需要参与到 rust 协作中.
  - 要缩短这个入门门槛, 才能解放生产力.
- rust 的入门门槛过高, 导致其他语言开发者, 无法短时间快速上手.
- 这是本教程的编写目的, 解决此问题.
